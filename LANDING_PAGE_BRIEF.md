## 2. 우리 서비스가 뭔지 — 핵심 요약

### 한 줄 요약
> **토플 독학 관리 프로그램. 강의 없이, 매일 스케줄대로 직접 풀고 직접 정리하는 시스템.**

### 운영자
- **이온쌤** 1인 운영 (직원 없음, 동업자 없음)
- 토종 한국인, 74점→119점 순수 독학으로 달성
- 이 사이트 + 테스트룸(학생용 풀이 사이트)도 직접 개발 (코딩 독학)

### 현재 운영 프로그램
| 프로그램 | 상태 | 설명 |
|----------|------|------|
| **내벨업챌린지** | **운영중** | 매일 스케줄에 따라 직접 풀고 직접 정리하는 독학 프로그램. Fast(4주)/Standard(8주) |
| **스라첨삭** | 준비중 | Speaking & Writing 첨삭 프로그램. 4월중 오픈 예정 |

### 핵심 차별점 (이 내용이 메인페이지 톤을 결정합니다)
1. **강의가 없다** — 선생님이 설명해주는 방식이 아님. 직접 풀고, 해설 읽고, 오답노트 쓰는 구조
2. **100% 컴퓨터 환경** — 실제 토플과 동일한 CBT 환경에서 연습 (종이 교재 X)
3. **강제성 시스템** — 매일 마감, 보증금, 인증률, 잔디 시스템, 주간체크
4. **해설이 강의를 대체** — 문장별 해석, 단어 팝업, 보기별 정오답 이유 전부 제공
5. **선별적 수강 심사** — 안 되는 학생은 거절함 ([승인불가] 실제로 많음)
6. **신청 → 무료 개별분석 → 승인 여부 결정** — 바로 결제가 아님
7. **시험료 지원금 21만원** — 31.5만원 시험을 16만원에

## 4. 변경 가능 / 불가능 범위 (중요)

### 변경 불가 (유지 필수)
| 항목 | 이유 |
|------|------|
| **Navigation (GNB) 전체** | 모든 페이지에서 공유하는 공통 컴포넌트. 메뉴 구조, 로고 위치, 드롭다운 모두 유지 |
| **Footer 전체** | 모든 페이지 공통 |
| **플로팅 버튼 3종** | 카카오톡(우하단), 관리자 대시보드(좌하단), 학생 대시보드(좌하단). 로그인 상태에 따라 JS로 동적 표시 |
| **JS 파일 3개 로드 순서** | `supabase-config.js` → `common.js` → `main.js` (이 순서와 연결 유지) |
| **CSS 변수 시스템** | `:root`에 정의된 컬러 변수들 (primary: `#9480c5`, accent: `#77bf7e` 등). 전체 사이트에서 사용중이므로 변수 자체는 건드리지 말 것 |
| **폰트** | Noto Sans KR (Google Fonts). 전역 적용 중 |
| **로고 이미지 경로** | `logo/black_logo.png` (검정 로고), `logo/white_logo.png` (흰색 로고) — GitHub Pages CDN 경로 사용중 |
| **반응형 breakpoint** | 768px 기준 모바일 전환 (기존 CSS 구조) |
| **Supabase 연동 (후기 섹션)** | `main.js`의 `loadReviews()` 함수가 DB에서 후기를 동적 로드. `#reviewsContainer` div는 유지 필요 |

### 변경 가능 (자유롭게 변경)
| 항목 | 설명 |
|------|------|
| **Hero 섹션 전체** | 헤드라인, 서브카피, CTA 버튼 문구, 통계 수치 — 전부 자유 변경 가능. 구조(레이아웃)도 변경 가능 |
| **About 섹션 전체** | "왜 이온토플인가요?" — 제목, 설명, 카드 개수, 아이콘, 내용 전부 변경 가능 |
| **Programs 섹션 전체** | 카드 디자인, 순서, 내용 변경 가능 (단, "스라첨삭 = 준비중"은 유지) |
| **Pricing 섹션 전체** | 가격 표시 방식, 강조점 변경 가능 |
| **Reviews 섹션 디자인** | 카드 디자인/레이아웃 변경 가능. 단, `id="reviewsContainer"`는 유지해야 JS가 여기에 후기를 넣음 |
| **CTA 섹션 전체** | 문구, 버튼, 배경 전부 변경 가능 |
| **섹션 순서** | 섹션 순서 변경 가능 |
| **섹션 추가/삭제** | 새 섹션 추가 가능, 불필요한 섹션 삭제 가능 |
| **인라인 CSS / 페이지 전용 스타일** | `index.html`에만 적용되는 스타일은 자유롭게 추가 가능. `<style>` 태그 또는 인라인 |
| **Hero 배경** | 현재 그라데이션 → 이미지/영상/일러스트 등으로 변경 가능 |
